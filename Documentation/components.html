Headers

Headers are the begninng block of the page.Headers provide page identity and global navigation, with search and perhaps other tools.

Header With Title

Plain header with just title element. 


<div class="header">
    Header
</div>

Header With Icons

Plain header with title and icons. 


<div class="header">
    <button class="button icon-button header-icon icon fa fa-bars"></button>
    <div class="header-title">Header</div>
    <button class="button icon-button header-icon icon fa fa-bars"></button>
</div>

Fixed Header

Header with fixed position at top of the page with other elements scrolling. Applying header-fixed class will fix your header. 


<div class="header fixed-header">
    <button class="button button-icon header-icon icon fa fa-bars"></button>
    <div class="header-title">Fixed Header</div>
    <button class="button button-icon header-icon icon fa fa-bars"></button>
</div>

List With Divider

Use item-divider class to create divider in any List.

    <li class="item">Item 1</li>

    <li class="item">Item 3</li>
    <li class="item">Item 4</li>

    <li class="item">Item 2</li>
