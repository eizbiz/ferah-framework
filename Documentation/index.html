About Ferah Framework

What is Ferah Framework

Ferah Framework is a fully open source solution for developing high-quality hybrid apps using the web technologies you love i.e. HTML, CSS and Javascript.

Ferah is designed to give you the best user experience for your hybrid apps. while developing Ferah we have been inspired by many open source frameworks and blogs supporting web technologies. Since it is build with pure HTML, CSS & JavaScript it fits in any space using web technologies i.e. Corodva/PhoneGap and any other framework using the same.

Ferah doesn't follow any specific design pattern instead we believe hybrid apps have a unique place in mobile app development environment and we should not have such limitation while developing UI components. We don't use customized html tags neither we use Angular JS or jQuery to complicate or library. And the best part is you can create apps for Android, IOS and Windows Phone using Ferah.

How is Ferah Framework Licensed?

Ferah is completely free and open source, released under the permissive MIT license, which means you can use it in personal or commercial projects for free.
